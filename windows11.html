Σύντομη Περιγραφή

Τα Windows 11 είναι μια σημαντική έκδοση του λειτουργικού συστήματος Windows NT που αναπτύχθηκε από τη Microsoft και ανακοινώθηκε στις 24 Ιουνίου 2021 και είναι ο διάδοχος των Windows 10, που κυκλοφόρησαν το 2015. Τα Windows 11 κυκλοφόρησαν στις 5 Οκτωβρίου 2021, ως δωρεάν αναβάθμιση μέσω του Windows Update για κατάλληλες συσκευές με Windows 10.

Κάποιες από τις σημαντικές αλλαγές που αφορούν τα Windows 11 είναι:

  • Task Bar
  • Start Menu
  • Widgets
  • Chat
  • Snap Layouts
  • User Interface
